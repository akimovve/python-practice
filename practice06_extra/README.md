### Задача 1

Напишите функцию deriv для приближенного вычисления производной в заданной
точке.

#### Решение

```bigquery
def deriv(func):
    x = sy.Symbol('x')

    def take(x):
        return sy.diff(func(x), x)

    def substitution(dig):
        return take(x).subs(x, dig)

    return substitution
```

#### Демо

```bigquery
>>> deriv(lambda x: x ** 3)(5)
75.00014999664018
```
